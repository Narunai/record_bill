ทำเว็ป รายรับรายจ่าย ให้ใช้ง่ายโดยมีการสมัครเข้าเว็ป ง่ายๆโดย gmail หรือ id และ รหัส
ให้สามารถดูบันทึกรายรับรายจ่ายเป็นวัน

บันทึกค่าใช้จ่ายต่างๆ โดยไม่ต้องใส่หน่วยก็ได้และ ให้สามารถเลือกตัวเลือกที่ใช้บ่อยได้ เพื่อความเร็วในการใช้และความสะดวก
ทำระบบหลังบ้านด้วย db python


  1. วิธีรัน Backend (Python API)
  เปิด Terminal หน้าต่างที่ 1 แล้วพิมพ์:
   1 cd backend
   2 python -m uvicorn app.main:app --reload
  ระบบจะทำงานที่: http://localhost:8000

  2. วิธีรัน Frontend (React Web)
  เปิด Terminal หน้าต่างที่ 2 แล้วพิมพ์:

   1 cd frontend
   2 npm run dev
  ระบบจะทำงานที่: http://localhost:5173


    สาเหตุที่เป็นไปได้:
   1. Session หมดอายุ: บางครั้งการเชื่อมต่อระหว่าง CLI กับเซิร์ฟเวอร์ของ Google อาจหลุดหรือหมดอายุ
   2. ปัญหาด้าน Subscription: บัญชีที่ใช้ระบุว่าจำเป็นต้องมีการสมัครใช้งาน (Subscription) หรือสิทธิ์ระดับ Enterprise

  วิธีแก้ไขเบื้องต้น:
   1. ลอง Login ใหม่: ให้คุณปิดหน้าต่างนี้แล้วรันคำสั่ง:

   1     gemini logout
   2     gemini login
   2. ตรวจสอบเวอร์ชัน: ตรวจสอบว่าคุณใช้ Gemini CLI เวอร์ชันล่าสุดหรือไม่ โดยรันคำสั่ง:

   1     npm install -g @google/gemini-cli
   3. ตรวจสอบสิทธิ์การใช้งาน: หากคุณใช้งานในระดับบริษัท (Enterprise) อาจต้องติดต่อผู้ดูแลระบบเพื่อขอสิทธิ์การใช้งานครับ

  ส่วนโค้ดล่าสุดที่ผมแก้ไขให้ (ปุ่มเลือกเดือนและปีแบบตาราง) ผมได้เขียนลงไฟล์ Dashboard.tsx เรียบร้อยแล้วครับ หากระบบกลับมาทำงานปกติ
  คุณจะเห็นหน้าตา Dashboard ที่มีปุ่มกดเลือกเดือนและปีที่สะดวกขึ้นเหมือนหน้าปฏิทินวันครับ

   แก้ไขการแสดงผลที่ตัวเลข + กันเยอะๆให้ตัดลงมาแถวต่อไปและให้เขียนตัวเลขรวมไว้ด้วย
